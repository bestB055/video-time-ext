这个文件夹的内容需要上传到 GitHub 仓库根目录。

上传后的 GitHub Pages 访问路径：
  update.xml  → https://你的用户名.github.io/video-time-ext/update.xml
  video-time-extension.crx → https://你的用户名.github.io/video-time-ext/video-time-extension.crx

操作步骤：
1. 先编辑 update.xml，填入真实的扩展ID和GitHub用户名
2. 把 Chrome 打包生成的 .crx 文件也放到这个文件夹
3. 将这个文件夹内的所有文件上传到 GitHub 仓库根目录
